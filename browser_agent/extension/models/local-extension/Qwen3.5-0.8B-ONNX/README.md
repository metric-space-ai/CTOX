# Qwen3.5-0.8B-ONNX Browser Agent Runtime

This folder defines the curated ONNX runtime and browser-training helper artifacts for the CTO-Agent Chrome extension.

Canonical hosted artifact repo:

- `metricspace/Qwen3.5-0.8B-ONNX-browser-agent`

The Git repo only keeps the manifests and integration metadata. Large ONNX payloads are published separately and fetched by the extension from Hugging Face.
