Main meta-skill: `read-trace-first`

Use inside CTOX:

- Read the current task from `Focus`.
- Use `Verified evidence` and `Anchors` to keep tracing relevant.
- This phase exists to find the exact later change point, not to solve the whole task yet.

- Read before editing.
- Trace one path at a time.
- Find the exact file and symbol to change.
- Find the smallest later verification step.
- Do not patch from guesses.
- If two paths still look equally likely, stop and say that.
