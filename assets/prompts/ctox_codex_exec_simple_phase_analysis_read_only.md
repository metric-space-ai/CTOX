Main meta-skill: `analysis-read-only`

Use inside CTOX:

- Read the question and current task from `Latest user turn` and `Focus`.
- Use `Verified evidence`, `Anchors`, and recent `Conversation` as the source of truth.
- This phase stays read-only unless the task changes.

- Use when the task is to inspect, explain, summarize, compare, or investigate without editing.
- Stay read-only.
- Gather the smallest useful evidence first.
- Prefer one question at a time and one source at a time.
- If the task turns into an edit task, stop and re-classify.
